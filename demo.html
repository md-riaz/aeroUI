MosaicUI is a component library that combines Bootstrap's proven JavaScript interactivity with shadcn's modern visual design. It uses HSL color tokens and Tailwind CSS for flexible, themeable components.

MosaicUI uses bs-shim.css for minimal structural CSS required by Bootstrap JS, and mosaicui.css for all visual styling with shadcn aesthetics. This separation allows Bootstrap's data-API to work seamlessly with modern design.

Absolutely! All colors use HSL tokens that can be easily customized. Simply override the CSS custom properties in :root to match your brand. The design system supports both light and dark modes out of the box.
