aeroUI is a component layer that keeps Bootstrap's JavaScript interactivity while applying shadcn visuals. It uses Tailwind utilities inside a single CSS file for flexible, token-driven components.

aeroUI compiles a single aeroui.css file with the Tailwind CLI. Structural classes for Bootstrap JS and visual styling live together, so Bootstrap's data-API works seamlessly with the shadcn token palette.

Absolutely! Everything is composed with Tailwind utilities, so you can swap tokens or adjust spacing directly in the source file to match your brand.
