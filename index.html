Why aeroUI

Minimal setup, major glow-up

Keep your existing Bootstrap behaviour while adopting Tailwind's utility-first workflow. No rewrites, no new APIs — just drop in the CSS and ship.

  • Bootstrap compatible. Works with bootstrap.bundle.min.js so dropdowns, modals, and accordions behave exactly as expected.
  • shadcn tokens built-in. Customise via CSS variables or Tailwind's theme extension for full design control.
  • Single source file. Tailwind handles purging and extension so you can add, remove, or tweak components quickly.

Install & build

Run the Tailwind CLI alongside your Bootstrap project to generate a purged build tailored to your markup. 

npm install
npm run dev

Import the layer

Keep your Tailwind source clean with a single import that exposes every aeroUI component. 

/* tailwind.css */
@tailwind base;
@tailwind components;
@tailwind utilities;

@import "./aeroui.css";
