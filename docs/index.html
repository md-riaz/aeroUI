What you get

  • Ready-to-copy Bootstrap markup styled with Tailwind design tokens.
  • Utility-driven tokens for spacing, typography, and color without configuring Tailwind.
  • Lightweight theme switcher and component variants you can extend as needed.

Quick start

Drop the compiled stylesheet next to your project assets and include it after Bootstrap. No package manager or build tooling required.

  1. 1. Copy dist/aeroui.css into your project.
  2. 2. Add the stylesheet after Bootstrap in your HTML head so aeroUI tokens can override defaults.
  3. 3. Paste components from the repository or the Blocks catalog and customize copy or icons as needed.

Include in your page

<link rel="stylesheet" href="https://cdn.jsdelivr.net/npm/bootstrap@5.3.3/dist/css/bootstrap.min.css">
<link rel="stylesheet" href="/path/to/aeroui.css">
<script defer src="https://cdn.jsdelivr.net/npm/bootstrap@5.3.3/dist/js/bootstrap.bundle.min.js"></script>

The CSS bundles Tailwind-inspired utility tokens and component styles. No JavaScript bundlers or npm scripts are required.
