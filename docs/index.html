Shadcn × Tailwind × Bootstrap motto

Build polished experiences with Tailwind-calibrated tokens.

aeroUI ships ready-to-copy markup wired to Tailwind utility recipes and shadcn/ui tokens while embracing Bootstrap’s familiar semantics and JavaScript behaviours. Copy the markup, compile src/aeroui.css with your Tailwind pipeline, ship an optimised stylesheet that matches your project, and load bootstrap.bundle.min.js to power interactive behaviours—no extra jQuery required.

Project Motto: Shadcn UI built on top of Tailwind CSS and the syntax, semantics, and JavaScript interactivity of Bootstrap for a single, cohesive toolkit.

Tailwind-powered authoring

Copy the markup, add src/aeroui.css to your Tailwind build, and let the compiler generate a production-ready bundle for your project.

# tailwind.config.js
content: [
  "./resources/views/**/*.blade.php",
  "./resources/js/**/*.js",
  "./vendor/aeroui/**/*.html"
],

# vite.config.js / tailwind CLI output
import "./resources/css/aeroui.css";
